🍿 Netflix EDA — Data Analysis Project








📊 Overview

Exploratory Data Analysis of Netflix Movies & TV Shows dataset to uncover trends, patterns, and insights using Python.

This project analyzes:

• 🎬 Movies vs TV Shows distribution


• 🌍 Country-wise content production


• 📈 Content growth over years


• 🎭 Genre popularity


• 📊 Real-world dataset cleaning & visualization





🛠 Tech Stack

Python


Pandas


NumPy
